# Community Detection Dashboard

This repository contains a graph topology analysis project for the CA-GrQc dataset. It includes a Jupyter notebook, a Python dashboard script, and graph community detection outputs.

## Contents

- `CDD.ipynb` - Main notebook for data loading, graph analysis, and community detection.
- `community_dashboard.py` - Python script for dashboard generation and visualization.
- `CA-GrQc.txt` - Source edge list for the collaboration graph.
- `ca-GrQc_LabelPropagation_Communities.png` - Label Propagation community result image.
- `ca-GrQc_Louvain_Communities.png` - Louvain community result image.
- `Documentation.docx` - Project documentation.
- `Mohamed_Hussein_Task2_Presentation.pptx` - Presentation slides.

## Usage

1. Open `CDD.ipynb` in Jupyter Notebook or JupyterLab.
2. Run the notebook cells in order.
3. Optionally run `community_dashboard.py` with Python to generate dashboards.
